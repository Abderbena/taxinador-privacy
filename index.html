Politique de confidentialité – Taxi Nador

Dernière mise à jour : 25/02/2026

1. Données collectées

1.1. Données que vous fournissez

L’Application ne nécessite pas de création de compte et ne collecte pas de données d’identification (nom, adresse, mot de passe, etc.). Vous pouvez toutefois envoyer volontairement des informations, par exemple : propositions de prix observés, suggestions de quartiers/hay, signalements de chauffeurs, etc.

1.2. Données collectées automatiquement

Nous utilisons Firebase Analytics (Google) pour collecter des statistiques anonymes d’utilisation (fonctionnalités utilisées, fréquence, type d’appareil, erreurs, etc.).

Lorsque vous utilisez la carte, l’Application peut accéder à votre position pour afficher votre position sur la carte et vous aider à visualiser votre itinéraire. Vous pouvez désactiver la localisation dans les réglages de votre appareil.

1.3. Stockage dans Firebase (Cloud Firestore)

Les contributions (prix, trajets, profils de chauffeurs, likes, signalements…) sont stockées dans Cloud Firestore (service fourni par Google) afin d’afficher des informations utiles aux autres utilisateurs et de lutter contre les abus.

2. Base légale du traitement

Les données sont traitées sur la base de votre consentement (localisation, envoi volontaire d’informations) et de notre intérêt légitime à améliorer l’Application et lutter contre les abus.

3. Partage des données

Nous ne vendons pas vos données. Elles sont partagées uniquement avec nos prestataires techniques (Google Firebase) et, le cas échéant, avec les autorités compétentes si la loi l’exige.

4. Vos droits

Vous pouvez demander l’accès, la rectification ou la suppression de certaines données vous concernant en nous contactant à l’adresse : taxi.nador.morocco2030@gmail.com.
