Politique de confidentialité – Application Taxi Nador Dernière mise à jour : 25/02/2026 L’application Taxi Nador (« l’Application ») est éditée par Taxi Nador. La présente politique de confidentialité explique quelles données sont collectées, comment elles sont utilisées et quels sont vos droits. 1. Données collectées 1.1. Données que vous fournissez L’Application ne nécessite PAS de création de compte et ne collecte pas de données d’identification (nom, adresse, mot de passe, etc.). Vous pouvez cependant envoyer volontairement des informations, par exemple : - des propositions de prix observés pour un trajet ; - des suggestions de quartiers/hay à ajouter ; - des signalements de chauffeurs. Ces informations peuvent contenir : - des descriptions textuelles ; - des informations sur un trajet (quartier, prix, jour/nuit) ; - des informations sur un chauffeur (plaque, zone, numéro de téléphone si vous le renseignez). 1.2. Données collectées automatiquement a) Données d’utilisation et statistiques (Firebase Analytics) Nous utilisons Firebase Analytics (service fourni par Google) pour collecter de manière anonyme des informations d’utilisation de l’Application, par exemple : - fonctionnalités utilisées ; - fréquence d’utilisation ; - type d’appareil et version Android ; - erreurs et plantages éventuels. Ces données sont utilisées uniquement pour améliorer l’Application (stabilité, ergonomie, fonctionnalités). b) Données de localisation (carte) Lorsque vous utilisez la carte dans Taxi Nador, l’Application peut accéder à votre position approximative ou précise (GPS) pour : - afficher votre position sur la carte ; - vous aider à visualiser votre itinéraire ou votre proximité avec certains points (quartiers, stations de taxis, etc.). La localisation est utilisée UNIQUEMENT à l’intérieur de l’Application pour ces fonctionnalités. Nous ne suivons pas vos déplacements en continu et nous ne revendons pas ces données. Vous pouvez désactiver à tout moment l’accès à la localisation dans les paramètres de votre appareil Android. Dans ce cas, certaines fonctionnalités de la carte pourront être limitées. 1.3. Stockage dans Firebase (Cloud Firestore) Lorsque vous contribuez (propositions de prix, enregistrement de chauffeurs, signalements), ces données sont stockées dans une base de données Firebase (Cloud Firestore) hébergée par Google. Les informations stockées peuvent inclure : - identifiant technique du trajet ou du chauffeur ; - prix proposés ; - plaque d’immatriculation et informations sur le véhicule ; - numéro de téléphone du chauffeur (si renseigné) ; - nombre de « likes » et signalements. Ces données sont utilisées pour : - afficher des informations utiles aux autres utilisateurs (prix moyens, chauffeurs, etc.) ; - améliorer la qualité des informations (détection d’abus, suppression de profils signalés). 2. Base légale du traitement Nous traitons vos données : - sur la base de votre **consentement** lorsque vous autorisez la localisation ou que vous envoyez volontairement des informations ; - sur la base de notre **intérêt légitime** à améliorer l’Application (statistiques anonymes, stabilité, lutte contre les abus). 3. Partage des données Nous ne vendons pas vos données. Les données sont partagées uniquement avec : - Google (Firebase Analytics et Cloud Firestore), en tant que prestataire technique ; - éventuellement avec des autorités compétentes si la loi nous y oblige (fraude, réquisition légale, etc.). 4. Durée de conservation Les données sont conservées pendant une durée raisonnable nécessaire aux finalités décrites ci‑dessus, par exemple : - contributions de prix et trajets : tant qu’elles sont pertinentes pour l’information des utilisateurs ; - profils de chauffeurs : tant que le chauffeur n’a pas demandé la suppression ou que le profil n’est pas désactivé ; - statistiques Firebase Analytics : selon les réglages par défaut de Google/Firebase. Vous pouvez nous contacter pour demander la suppression ou la modification d’informations vous concernant. 5. Sécurité Nous mettons en œuvre des mesures raisonnables pour protéger les données (accès restreint à la base de données, authentification par compte développeur, etc.). Cependant, aucun système n’est parfaitement sécurisé et nous ne pouvons pas garantir une sécurité absolue. 6. Vos droits Selon votre pays de résidence, vous pouvez disposer des droits suivants : - droit d’accès : savoir quelles données sont stockées à votre sujet ; - droit de rectification : corriger des informations inexactes ; - droit de suppression : demander la suppression de certaines données ; - droit d’opposition ou de limitation du traitement dans certains cas. Pour exercer vos droits, contactez‑nous à l’adresse suivante : info.morocco2030@gmail.com 7. Services tiers L’Application utilise les services tiers suivants : - Firebase Analytics (Google) – statistiques anonymes d’utilisation ; - Firebase Cloud Firestore (Google) – stockage des données de trajets, contributions, chauffeurs ; - API de cartographie et géocodage (OpenStreetMap / OsmDroid / services associés) – affichage de la carte et calcul d’itinéraires. Ces services peuvent collecter certaines données selon leurs propres politiques de confidentialité : - Google Firebase : https://policies.google.com/privacy - OpenStreetMap : https://wiki.osmfoundation.org/wiki/Privacy_Policy Nous vous invitons à consulter ces politiques pour plus de détails. 8. Enfants L’Application n’est pas spécifiquement destinée aux enfants. Si vous pensez que des données concernant un mineur ont été enregistrées, contactez‑nous pour demander leur suppression. 9. Modifications de cette politique Nous pouvons mettre à jour cette politique de confidentialité pour refléter des changements de l’Application, des services tiers utilisés ou de la réglementation. La date de dernière mise à jour est indiquée en haut du document. En continuant d’utiliser l’Application après une mise à jour, vous acceptez la version modifiée de la politique. 10. Contact Pour toute question concernant cette politique de confidentialité ou vos données, vous pouvez nous contacter : Taxi Nador info.morocco2030@gmail.com 25/02/2026
